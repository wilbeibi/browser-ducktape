# Browser Ducktape 🦆

Userscripts for patching up daily browsing annoyances.

## Install

1. Get [Violentmonkey](https://violentmonkey.github.io/) — [Chrome](https://chrome.google.com/webstore/detail/violentmonkey/jinjacmigcnfinphajbemlabodjabnV) / [Firefox](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)
2. Open any `.js` file here, click **Raw**, and Violentmonkey will pick it up.

## Scripts

### 🧠 Focus & Accessibility
- **`adhd_reader.js`** — Reading ruler that highlights the line under your cursor.
  ![Reading Ruler Demo](screenshots/highlight.png)

- **`worth_watching.js`** — Blurs YouTube/Bilibili and makes you confirm before watching.

### 🤖 AI Tools
- **`claude_usage_pace.js`** — Adds a pace marker to the Claude.ai usage bar so you don't burn through messages too fast.
  ![Claude Usage Pace Demo](screenshots/pacer.png)

- **`gemini_dynamic_tab_title.js`** — Updates the tab title to match the current Gemini conversation.

### 🛠️ Utilities
- **`deepwiki_on_github.js`** — Adds a DeepWiki button to GitHub repo pages.
  ![DeepWiki on GitHub Demo](screenshots/deepwiki.png)

- **`youtube_transcript_downloader.js`** — Download button for the YouTube transcript panel.
